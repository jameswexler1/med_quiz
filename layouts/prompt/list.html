🇧🇷 Português

Crie 10 questões de múltipla escolha para residência médica sobre o tema: [TEMA AQUI].

Requisitos:
- Cada questão deve ter 5 alternativas (A, B, C, D, E)
- Nível de dificuldade: residência médica (difícil)
- Inclua explicação detalhada da resposta correta
- Mencione mecanismo fisiopatológico quando relevante
- Use termos médicos precisos em português

Retorne SOMENTE um array JSON válido, sem texto antes ou depois, sem markdown, sem blocos de código. Use exatamente este formato:

[
  {
    "title": "Texto completo da pergunta aqui?",
    "choices": [
      "A) Primeira alternativa",
      "B) Segunda alternativa",
      "C) Terceira alternativa",
      "D) Quarta alternativa",
      "E) Quinta alternativa"
    ],
    "correctIndex": 2,
    "explanation": "Explicação detalhada da resposta correta, incluindo fisiopatologia e pontos-chave."
  }
]

Lembre-se: correctIndex é o índice (0-4) da alternativa correta no array choices.

Create 10 multiple-choice questions for medical board/residency exam on the topic: [TOPIC HERE].

Requirements:
- Each question must have 5 answer choices (A, B, C, D, E)
- Difficulty level: residency/board exam (hard)
- Include a detailed explanation for the correct answer
- Mention the pathophysiological mechanism when relevant
- Use precise medical terminology

Return ONLY a valid JSON array, with no text before or after, no markdown, no code blocks. Use exactly this format:

[
  {
    "title": "Full question text here?",
    "choices": [
      "A) First option",
      "B) Second option",
      "C) Third option",
      "D) Fourth option",
      "E) Fifth option"
    ],
    "correctIndex": 2,
    "explanation": "Detailed explanation of the correct answer, including pathophysiology and key teaching points."
  }
]

Remember: correctIndex is the 0-based index (0-4) of the correct answer in the choices array.
